# TECHNOLOGIES & TOOLS USED

## Core Technologies

### 1. **Programming Language**
- **Python 3.9+**

### 2. **LLM Providers & APIs**
- **OpenAI GPT-4** - High-quality review generation
- **OpenAI Batch API** - Cost-effective batch processing (50% discount)
- **Google Gemini 2.0 Flash Exp** - Fast generation & LLM judge evaluation

### 3. **Orchestration & Workflow**
- **LangGraph** - Multi-agent system orchestration
- **LangChain** - LLM integration utilities

### 4. **Prompt Engineering & Optimization**
- **DSPy** - Structured prompt management, optimization, and type-safe signatures

### 5. **Data Generation & Randomization**
- **Faker** - Generate random personas, seed words, and diverse data

### 6. **Quality Metrics & NLP**
- **NLTK** - Self-BLEU calculation (diversity metric)
- **Sentence-Transformers** - Semantic similarity (embeddings)
- **Transformers (HuggingFace)** - Sentiment analysis, perplexity calculation
- **Scikit-learn** - Cosine similarity, statistical analysis
- **SciPy** - Chi-square tests for bias detection

### 7. **Data Processing**
- **Pandas** - Data manipulation and analysis
- **NumPy** - Numerical computations
- **JSONLines** - Dataset storage format

### 8. **Configuration Management**
- **PyYAML** - Configuration file parsing
- **python-dotenv** - Environment variable management

### 9. **CLI & API**
- **Click** - Command-line interface
- **FastAPI** - REST API endpoints (optional)
- **Uvicorn** - ASGI server
- **Pydantic** - Data validation

### 10. **Logging & Monitoring**
- **Python logging** - Standard library logging
- **JSON** - Structured log storage
- **Rich** - Terminal formatting and progress bars
- **tqdm** - Progress tracking

### 11. **Testing**
- **pytest** - Unit and integration testing
- **pytest-asyncio** - Async testing support

### 12. **Deep Learning Framework**
- **PyTorch** - Backend for transformers/embeddings
